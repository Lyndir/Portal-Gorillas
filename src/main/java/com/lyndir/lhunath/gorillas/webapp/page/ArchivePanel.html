Welcome to the Gorillas Archives. Here you'll find an overview of the current and past Gorillas versions.

Note

The Gorillas code uses a slightly modified version of the Cocos2D-iPhone code. These modifications are mainly bug-fixes and the addition of slow-motion capability (as used when pausing the game or showing kill-shots).

Due to this, you need to make sure to use both the Gorillas code at a certain version and my Cocos2D-iPhone snapshot from that same version for a quick and painless start. So, either download the snapshots of a specific version or check a version tag out of GIT with git checkout [gorillas-version] in your Gorillas GIT repository and git checkout gorillas-[gorillas-version] in your Cocos2D-iPhone GIT repository.

You are, of course, welcome to scour the rest of the code-base and branch for anything you like. Should you have any further troubles, please first think them over a few more times, and contact me if you can't dig your way out of it.
